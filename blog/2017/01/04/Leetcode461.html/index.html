leetcode461. Hamming Distance

发表于 2017-01-04 | 分类于 Algorithm | 阅读次数

位操作是程序设计中对位模式按位或二进制数的一元和二元操作，位运算其实就是直接对在内存中的二进制数据进行操作，因此处理数据的速度非常快。更详细介绍参考这里

The Hamming distance between two integers is the number of positions at which the corresponding bits are different.
Given two integers x and y, calculate the Hamming distance.（0 ≤ x, y < 2^31）

这道题目看明白题目，其实非常简单，就是对两个数进行异或，然后判断异或结果后二进制表示中1的个数。可能存在的难点是如何判断一个数（n）的二进制表示中1的个数，可以仍然使用位操作，每次计算n和n-1的与值，指导结果为0，循环次数即为n中1的个数。

传送门：leetcode258
