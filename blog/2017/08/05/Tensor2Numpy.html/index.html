TensorFlow 之 tensor转numpy

发表于 2017-08-05 | 分类于 Machine Learning ， Deep Learning | 阅读次数

毕设第一次接触TensorFlow，不得不说，作为google的“亲儿子”，个人觉得TensorFlow的使用体验真的有些差，官方文档编排的有些混乱，不是很容易梳理结构，API的参数说明也有一些不清不楚，实际编程中debuge过程很难过。。。总之，感觉TensorFlow对于我这样的新手菜鸟不是非常友好。。。
