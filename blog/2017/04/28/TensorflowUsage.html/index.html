TensorFlow 使用记录

发表于 2017-04-28   |   分类于 Machine Learning ， Deep Learning   |     |   阅读次数

记录在使用TensorFlow过程中遇到的一些问题，不断更新…

1、Session()与InteractiveSession()

  • 创建Session对象，用于启动图，Session对象在使用完后需要关闭以释放资源. 除了显式调用 close 外, 也可以使用 “with” 代码块 来自动完成关闭动作。

  • 为了便于使用诸如IPython之类的Python 交互环境，可以使用InteractiveSession 代替 Session类，使用Tensor.eval()和Operation.run() 方法代替Session.run()。这样可以避免使用一个变量来持有会话。

2、reshape()操作

函数将张量tensor变换为参数shape的形式，shape是一个列表形式，列表中可以存在-1，-1代表不需要人为指定这一维度的大小，函数会自动计算，但列表中只能存在一个-1。

3、gather()和gather_nd()

gather函数原型：
gather(params, indices, validate_indices=None, name=None)
函数根据indices对params进行切片，indices必须是一个任意维度的整型tensor（通常为标量或向量），返回大小为indices.shape + params.shape[1:]的tensor

gather_nd函数原型：
gather_nd(params, indices, name=None)
函数也是根据indices对params进行切片，但是indices必须是一个shape为[d0, …, d{Q-2}, K]的整型tensor，其中P为params的最大下标，Q为indices的下标，0
