TensorFlow 之 卷积层Padding

发表于 2017-04-29 | 分类于 Machine Learning ， Deep Learning | 阅读次数

在使用TensorFlow搭建卷积神经网络时，卷积层函数

tf.nn.conv2d(input,filter,strides,padding,use_cudnn_on_gpu=None,data_format=None,name=None)中的参数padding指明零填充的算法，有两种方式可供选择：SAME, VALID。
为了方便对两种零填充算法的说明，定义以下符号：

  • 输入数据尺寸 W × H x D
  • 滤波器个数 K
  • 滤波器filter空间尺寸 F×F（卷积核）
  • 步长 stride值 S
  • 输出数据尺寸 out_W x out_H x out_D

1、padding = ‘SAME’

SAME 代表保留边界处的卷积结果，通常会导致输出shape与输入shape相同。

2、padding = ‘VALID’

VALID 代表只进行有效的卷积，即对边界数据不处理。
