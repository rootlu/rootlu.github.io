1、set.count()

set 是STL中的一种关联式容器，所谓关联容器就是通过键（key）来读取和修改元素，在set中的每个元素的值是唯一的，而且系统根据元素的值自动排序。set与map不同，它只是键（key）的集合。
set中的count函数用于计数某个数在set中出现的次数，但由于set的特殊性，该函数的返回值为0或者1，所以常用来判断set中是否含有某个元素.
函数原型：size_type count (const value_type& val) const;
