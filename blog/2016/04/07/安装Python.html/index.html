安装 Python

发表于 2016-04-07   |   分类于 Python   |     |   阅读次数

Python是一门简单易学，功能强大的解释性语言，它没有编译和链接，但具有高效的高级数据结构和简单而有效的面向对象编程方法。Python优雅的语法和动态类型以及其解释性的性质，使它在许多领域和大多数平台成为编写脚本和快速应用程序开发的理想语言。

1、下载Python

学习Python，首先需要把Python安装到电脑中，目前Python有两个版本，一个是2.x版本，一个是3.x版本，这两个版本有不兼容的，存在很大区别，用官方的介绍为‘Python 2.x is legacy, Python 3.x is the present and future of the language’!，因此现在Python正在向3.x进化，但到目前为止，仍然还有一些第三方的库不能使用，同时，针对Python 2.x的相关资料更全面，为方便学习使用，建议安装2.x版本。当然，具体版本还要根据需求确定。
我的电脑（windows）上安装的是Python2.7版本，所以下面的一系列教程均以2.7版本为基础，Python2.7版本可以在官网上找到，如下图所示。

2、安装Python

官网下载下来的安装程序是.msi格式的，双击即可安装。
安装过程中需要注意的是，在选择安装组件时，最好所有组件都勾选安装，特别是pip组件和Add python.exe to Path。pip是python中的包管理工具，python中许多包的安装都可以通过pip命令在线便捷安装，具体使用方法后续内容会讲到。Add python.exe to Path是自动添加python路径到环境变量中，简便我们手动添加环境变量的麻烦。

3、测试Python

python默认会安装到C:\Python27目录下，安装完成后，打开命令提示符窗口，敲入python，界面如下图所示，即表示python已经安装成功啦。

可能出现的错误
命令行输入python后，返回‘python’不是内部或外部命令，也不是可运行的程序或批处理文件。，说明python的路径没有添加到环境变量中。有两种解决办法，一种是自己手动添加python.exe的路径到Path环境变量中，一种是重新运行安装程序，即.msi文件，切记在选择组件时，勾选Add python.exe to Path。
