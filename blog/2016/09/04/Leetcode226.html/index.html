leetcode226. Invert Binary Tree

发表于 2016-09-04   |   分类于 Algorithm   |     |   阅读次数

反转二叉树，根节点不变，交换左右子树。递归实现，递归出口为到达叶子结点。

Invert a binary tree.

两种实现方式。一种递归，一种非递归实现。、
方法一：递归实现
左右子树递归调用反转函数，交换左右子树，递归出口为到达叶子结点。

方法二：非递归实现
使用栈，先存入左子树，然后存入右子树，交换做左右子树，右子树出栈，循环直到到达叶子结点。
