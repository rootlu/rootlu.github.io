leetcode389. Find the Difference

Posted on 2016-09-04 | In Algorithm | Visitors

找到两个字符串中不同的唯一一个字符。

Given two strings s and t which consist of only lowercase letters.
String t is generated by random shuffling string s and then add one more letter at a random position.
Find the letter that was added in t.

Example:

第一想法是使用map<char,int>计数每个字符的个数，分别遍历一次s串和t串，对于s串，计数加一，对于t串，计数减一，返回map中的值不为0的键。
但是，很不辛，超时了。。。

第二种方法参考了网上的做法，使用vector(26,0)，遍历字符串s，在对应字符位置+1，遍历字符串t，在对应字符位置-1，返回vector中值不为0的下标+‘a’

传送门：leetcode389

写的还不错？那就来个红包吧！
