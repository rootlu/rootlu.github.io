Docker 实例

几个Docker简单实例。

1、文件系统解析

1.1 查看容器层次对应的存储ID

/var/lib/docker/image/aufs/layerdb/mounts/[主机ID]# cat mount-id
返回 [存储ID]

1.2 容器各层存储位置：

/var/lib/docker/aufs/diff 中ID=[存储ID]的即为对应的容器各层

1.3 新建文件abc

abc 存储在
/var/lib/docker/aufs/diff/[存储ID]/root
或
/var/lib/docker/aufs/mnt/[存储ID]/root

1.4 导出镜像

docker save -o ubuntu.tar ubuntu:14.04

1.5 查看导出镜像各层

tar -tf ubuntu.tar

2、 Dockerfile文件创建镜像

2.1 搭建tomcat环境

Dockerfile文件

tomcat7.sh脚本文件

创建镜像
docker build -t tomcat:1.0 .

启动容器
docker run -d -p 8090:8080 --name tomcatTest tomcat:1.0

3、网络配置

  • docker run -d -P --name web training/webapp python app.py 随机映射到49000~49900端口 ip:hostPort:containerPort
  • docker ps 查看端口映射情况
  • docker logs -f web 查看应用信息
  • docker port web 查看端口映射情况

4、数据卷

docker run -it --name testV -v /home/luyf/work/testV:/root/testV ubuntu /bin/bash
加载主机目录/home/luyf/work/testV到容器的/root/testV目录下（目录若不存在，自动创建）
默认权限为读写
增删改文件，另一目录实时同步
容器删除，本地数据仍然存在。/var/lib/docker/volumes/[挂载点ID]/_data

5、数据卷容器

5.1创建数据卷

  • docker run -t -i -v /dbdata --name dbdata ubuntu 创建容器并挂载数据卷
  • docker run -t -i --volumes-from dbdata --name db1 ubuntu
  • docker run -t -i --volumes-from dbdata --name db2 ubuntu

5.2 备份

  • docker run --volumes-from dbdata -v $(pwd):/backup --name worker ubuntu tar cvf /backup/backup.tar /dbdata
    创建容器worker，挂载数据卷dbdata，挂载本地当前目录到worker的/backup目录
  • docker run -t -i -v /dbdata --name dbdata2 ubuntu /bin/bash
    docker run --volumes-from dbdata2 -v $(pwd):backup ubuntu tar xvf /backup/backup.tar
    数据恢复到dbdata2容器中

6、MySql数据存储

6.1 不挂载本地目录

  • 拉取官方镜像docker pull mysql
    -创建容器 docker run --name mysql -e MYSQL_ROOT_PASSWORD=121212 -d daocloud.io/library/mysql:5.7.14
  • 进入容器docker exec -it mysql bash
  • 登陆数据库mysql -uroot -p ,输入密码
  • 创建数据库create database testDB;
  • 创建表create table testTable (id int(4),name varchar(20),sex char(1));
  • 显示表结构describe testTable
  • 插入数据
    数据默认存储在/var/lib/mysql目录下
    停止容器，重启，数据仍然存在
    删除容器，数据丢失

6.2 挂载本地目录

创建容器，从宿主机挂载 /home/luyf/work/mysql/datadir 目录至容器内作为 /var/lib/mysql 目录，MySQL 就会默认将数据文件写入这个目录中。
docker run --name mysql -v /home/luyf/work/mysql/datadir:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=121212 -d daocloud.io/library/mysql:5.7.14

删除容器，数据仍然可以在本地找回

7.1 DockerUI

docker run -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock daocloud.io/rootlu/dockerui
