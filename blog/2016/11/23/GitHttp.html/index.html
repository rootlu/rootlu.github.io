Git https免密码push

发表于 2016-11-23 | 分类于 Git |

熟悉git的人都知道，每次使用git push提交代码时候，都需要输入密码，极简主义根本忍不了！！！针对git提交代码，有两种方式，一种是以ssh协议的方式，用ssh公钥设置免登陆，这种方式官网有详细说明。还有一种方式是https协议提交代码，下面主要介绍https协议如何实现push免密码。

进入项目目录

cd learn_git

  • 我这里的项目名称为learn_git

查看config文件

vim .git/config

修改配置文件

找到[remote]下面的url参数，修改为http://用户名:密码@github.com/rootlu/learn_git.git

写的还不错？那就来个红包吧！
