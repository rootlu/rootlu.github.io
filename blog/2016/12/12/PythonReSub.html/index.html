Python 之 re.sub用法

发表于 2016-12-12 | 分类于 Python | 阅读次数

正则表达式是Python的一大亮点，其功能非常强大，使用也很方便。在Python的标准库中包含了re模块，该模块提供了正则表达式匹配操作，正则表达式的具体使用方法，在此不再赘述，具体讲解可以参考转载博文《Python 之 正则表达式入门—转载》

在re模块中，定义了几个函数、常量和异常，其中使用最多的是re.compile(pattern, flags=0)，将正则表达式模式编译成一个正则表达式对象，它可以用于匹配使用它的match ()和search ()方法。在该模块中，还有一个功能强大的函数：re.sub，它可以实现正则的替换，下面将详细介绍该函数。

1、函数定义

函数的定义式为：re.sub(pattern, repl, string, count=0, flags=0)，官方文档对该函数的说明是：Return the string obtained by replacing the leftmost non-overlapping occurrences of pattern in string by the replacement repl.，翻译一下，函数利用正则表达式，实现复杂字符串的替换操作，函数返回替换后的字符串。

2.2 参数 repl

将要被替换的字符串，也可以是函数。如果repl是函数，它会被每个非重叠的正则调用，函数采用单个匹配对象作为参数，并返回替换字符串。
